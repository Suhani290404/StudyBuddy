# 📚 StudyBuddy AI

StudyBuddy AI is an intelligent study assistant that allows users to upload PDF notes and ask questions using AI.

## 🚀 Features (Planned)

- Upload PDF documents
- AI-powered question answering
- Semantic search using FAISS
- Gemini LLM integration
- Chat history
- Modern Streamlit interface

## 🛠️ Tech Stack

- Python
- Streamlit
- LangChain
- FAISS
- Hugging Face Embeddings
- Google Gemini API
